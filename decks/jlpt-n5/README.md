# JLPT N5

Vocabulary for the Japanese-Language Proficiency Test, level N5 (the most basic).

This folder is intentionally a placeholder. It needs:

- ~600–800 N5 vocabulary cards split across themed `.md` files
- Curated example sentences (consider [Tatoeba](https://tatoeba.org), CC-BY 2.0 FR)
- Native-speaker audio (planned, see [`docs/audio-contributions.md`](../../docs/audio-contributions.md))
